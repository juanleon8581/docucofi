# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-20

### ✨ Added

- **Templates System**: Core registry and navigation for templates.
- **Template Card**: High-quality `TemplateCard` component with integrated Vitest tests.
- **New Template**: Registered `cuenta-de-cobro` (Invoice for professional services) template.
- **I18n**: Comprehensive multi-language support (ES/EN) for templates and navigation.

### 🔄 Changed

- **Navigation Redesign**: Updated sidebar navigation with improved layout and template access.
- **System Architecture**: Refactored route groups and layout into an adaptive root structure for better scalability.
- **Templates Page**: Modernized templates overview page using newly developed UI components.

### 🎨 Styling

- **UI Components**: Integrated shadcn `Card` component for professional-grade interface design.

### 🔧 Chore & Tooling

- **Project Config**: Optimized AI assistant settings for local development.
